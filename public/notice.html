[ 공지사항 ]

2023년 6월 30일

말씀해주신 수정사항을 반영하고 반영된 사항을 날짜별로 알려드리고 있습니다.

앞으로도 많은 관심과 사랑 부탁드립니다. 😀

✅ 피드백 주신 내용으로 반영된 사항

  • 로그인한 상태로 회원가입, 로그인 페이지 접근 불가
  • 회원가입시 유효성검사 추가 및 올바르지 않은 형식인 경우 중복확인 검사 차단
  • 회원탈퇴 기능
  • 테스트 기간동안 여행이 종료되지 않아도 리뷰 작성 가능
  • 여행에 참여하지 않은 경우 후기 입력창 disable 처리
  • 모임 신청 제한시 명확한 이유 출력
  • [Style] cursor: pointer 추가
  • 탈퇴한 회원 게시글 삭제
  • 채팅방 재입장시 이전 대화내용 출력
  • 마이페이지 게시글에서 게시글간 구분

⭐️ 반영 예정 사항

  • PrivateRoute
  • 회원가입 페이지에 소셜로그인 기능 추가
  • 소셜로그인시 추가 정보 입력 1회로 제한
  • 소셜로그인 약관 전체동의시 선택동의 자동 체크
  • 로그아웃 실패시 동작
  • 전체적인 반응형 작업
  • 모집완료 게시글의 신청하기 버튼 -> 모집완료 버튼
  • 마이페이지 사용자 정보 수정(프로필 이미지, 닉네임, 소개 내용)
  • 마이페이지 리뷰/게시글 placeholder
  • 채팅방 주최자 확인 기능
  • 상대방 프로필 조회
  • 신청 취소
  • 메인페이지 로고, 배너, 아이콘 사라지는 문제
  • 검색시 지역/인원수 드롭다운
